# Title of problem
##### Difficulty: Moderate

## Problem statement
You will be given a string `s`, your job is to find and return the longest palindromic substring in `s`. 

#### Expected output
s = "babad"

Output : "bab" - even "aba" is a valid output

-----------------------------------------------
s = "cbbc"

Output : "bb"
